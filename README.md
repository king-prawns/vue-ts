# vue-ts-rxjs

Vue, TypeScript and RxJS.

## Requirements

- Node >= v6.x
- Yarn >= v1.1 | NPM >= v5.0


## Available tasks

```sh

# Runs development server (Webpack dev server)
$ yarn dev

# Build command
$ yarn build

# Lint with TSLint
$ yarn lint

# Run only type checking
$ yarn tc

# Run unit tests (ava-ts)
$ yarn test

# Runs http-server on port 8082
$ yarn httpserver

```
